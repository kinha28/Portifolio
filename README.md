# 💻 Meu Portfólio Front-end

## 🇧🇷 Português

Portfólio desenvolvido com HTML, CSS e JavaScript, criado com o objetivo de apresentar meus projetos, evolução prática e habilidades em desenvolvimento front-end.

### ✨ Funcionalidades

* Layout responsivo para desktop e mobile
* Menu de navegação com links internos
* Carrossel de imagens dos projetos
* Modal de ampliação de imagens
* Navegação entre imagens com setas
* Botão flutuante do WhatsApp
* Animações e efeitos visuais em CSS
* Links para projetos publicados no GitHub Pages

### 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Git e GitHub

### 📱 Responsividade

O projeto foi desenvolvido com foco em adaptação para diferentes tamanhos de tela, incluindo notebooks e dispositivos Android.
